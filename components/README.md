## Components Map

This directory documents the four major parts of the project without moving
code or history. Each subfolder points to the existing implementation.

- `luminance-mcp/`: HTTP MCP wrapper over Luminance API
- `salesforce-mcp/`: Salesforce MCP integration notes/stubs
- `ai-insights-ui/`: Luminance AI Insights demo UI changes
- `agentic-layer/`: Agentic orchestration and Prismatic docs
